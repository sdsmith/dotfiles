# Installation

```sh
git clone <repo_link> ~/.dotfiles

# For full suite (zsh based)
./zsh_install

# For basic tools et al
./install
```

Note: `install` is a subset of what's configured by `zsh_install`, and `zsh_install` depends on `install`.
